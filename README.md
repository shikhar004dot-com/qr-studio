# QR Studio

A modern QR Code Generator web application built using HTML, CSS, and JavaScript.
## Features
- Generate QR from text or URLs
- Upload image to Cloudinary and generate QR from image URL
- Async image upload handling
- Loading indicators and user feedback
- Download QR as PNG
- Clean responsive UI

## Tech Stack
- HTML
- CSS
- JavaScript
- Cloudinary API
- QRCode.js
